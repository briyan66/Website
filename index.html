Ini adalah dokumentasi proyek Laravel saya menggunakan FluxUI.

Langkah Instalasi

  1. Install dependency: composer install && npm install
  2. Copy file .env dan konfigurasi database
  3. Generate key: php artisan key:generate
  4. Jalankan migrasi: php artisan migrate
  5. Jalankan server: php artisan serve

Database Migrations


// 1. Customers
Schema::create('customers', function (Blueprint $table) {
  $table->id();
  $table->string('name');
  $table->string('email')->unique();
  $table->text('address')->nullable();
  $table->timestamps();
});

// 2. Orders
Schema::create('orders', function (Blueprint $table) {
  $table->id();
  $table->foreignId('customer_id')->constrained('customers');
  $table->date('order_date');
  $table->decimal('total_amount', 10, 2)->default(0.00);
  $table->enum('status', ['pending', 'processing', 'completed', 'cancelled'])->default('pending');
  $table->timestamps();
});

// 3. Order Details
Schema::create('order_details', function (Blueprint $table) {
  $table->id();
  $table->foreignId('order_id')->constrained('orders');
  $table->foreignId('product_id')->constrained('products');
  $table->unsignedInteger('quantity')->default(1);
  $table->decimal('unit_price', 10, 2);
  $table->decimal('subtotal', 10, 2);
  $table->timestamps();
});

CRUD Produk dengan FluxUI

  • Route: /dashboard/products
  • Controller: ProductController.php
  • Fitur: Tambah, Lihat, Edit, Hapus Produk
  • Komponen FluxUI: Form, Table, Button, Notification

Validasi menggunakan Laravel Request dengan aturan seperti nama produk wajib diisi dan harga tidak boleh negatif.
